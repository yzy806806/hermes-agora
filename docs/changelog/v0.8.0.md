## v0.8.0 (2026-06-09)

### Added
- **Observability (Phase 8.1)**: `coordinator/observability/` — Prometheus
  metrics exporter, structured event logging, OpenTelemetry-compatible trace
  context propagation; `storage/events.py`, `storage/global_store.py`,
  `storage/storage_manager.py`
- **Multi-tenant Infrastructure (Phase 8.2)**: `coordinator/tenant/` — Tenant
  models, guard (quota/enforcement), manager (lifecycle), router (API

- **Dashboard Frontend (Phase 8.3)**: `dashboard.py` (API endpoints),
  `static/dashboard.html` (SPA), `static/dashboard.js` (real-time WS updates)
- **Integration & Testing (Phase 8.4)**: test_observability, test_multi_tenant,
  test_dashboard, test_phase8_integration, test_ws_tenant_lazy_init;
  `docs/DESIGN-phase8.md`

  ROADMAP.md
