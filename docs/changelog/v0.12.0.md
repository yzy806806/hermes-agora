## v0.12.0 (2026-06-11)

### Added
- **Phase 12.1 Agent SDK (Python)**: Standalone pip package `agora-agent-sdk`
  with AgoraAgentClient, AbstractBridge ABC, ToolAdapter, SessionStore, and
  run() event loop (`packages/agora-agent-sdk/`)
- **Phase 12.2 Hermes Bridge**: Daemon process connecting Hermes profiles to
  Agora via SDK, translating kanban↔WS messages (`packages/hermes-bridge/`)
- **Phase 12.3 CLI Bridge**: PTY subprocess manager + ToolAdapter for Codex,
  Claude Code, and OpenClaw agents (`packages/cli-bridge/`)
- **Phase 12.4 Node.js SDK**: npm package `@agora/agent-sdk` with TypeScript
  AgoraAgentClient (`packages/agora-agent-sdk-js/`)
- **Phase 12.5 Session Persistence**: SessionRecord storage model, CRUD API
  endpoints (POST/GET /sessions), project artifact KV store
  (GET/PUT/DELETE /projects/{id}/artifacts/{key})
- **Phase 12.6 Production Hardening**: E2E WebSocket test fixes, version
  assertion updates
- **Design Doc**: Phase 12 architecture document (`docs/DESIGN-phase12.md`)

### Changed
- Documentation updated: ARCHITECTURE.md (v0.12.0, Phase 12 sections),
  API.md (Session/Artifact/SDK API), ROADMAP.md (Phase 12 complete,
  Phase 13 planned)
